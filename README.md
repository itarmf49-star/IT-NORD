# IT NORD — موقع الحلول التقنية

موقع عصري لشركة IT NORD مع قاعدة بيانات SQLite ولوحة تحكم.

## المميزات

- **قاعدة بيانات SQLite** — خزن دائم للمحتوى
- **خادم Python (Flask)** — API وخدمة الموقع
- **لوحة تحكم بدون تسجيل دخول** — إدارة مباشرة
- **صفحة منفصلة لكل مشروع** — هوائيات، شبكات، سيرفرات، كاميرات، مباني ذكية
- **صور وفيديوهات** — معارض صور وروابط فيديو تعليمية

## التشغيل

```bash
pip install -r requirements.txt
python app.py
```

افتح: **http://127.0.0.1:5000**  
لوحة التحكم: **http://127.0.0.1:5000/admin**

## إضافة المشاريع المفقودة

```bash
python seed_projects.py
```

## رفع على GitHub

```bash
git add .
git commit -m "IT NORD site"
git push origin main
```

> **ملاحظة:** مجلد `data/` قد يحتوي قاعدة البيانات. احذفه من `.gitignore` إذا أردت رفعها.

## الملفات

| الملف | الوصف |
|-------|-------|
