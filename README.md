# IT NORD — موقع الهوائيات والتقنيات

موقع عصري وفاخر لشركة IT NORD مع لوحة تحكم كاملة.

## المميزات

- **الصفحة الرئيسية**: تصميم فاخر بألوان ذهبية وواجهة احترافية
- **أزرار التواصل**: هاتف، واتساب، بريد، موقع — بتصميم متناسق مع الموقع
- **لوحة التحكم**: تسجيل دخول آمن لإدارة المحتوى
- **إدارة الوسائط**: تغيير الصور والفيديو في أي قسم
- **إدارة الأدوات**: إضافة وحذف أدوات وتقنيات
- **البطاقات المميزة**: تعديل الصور والعناوين في الصفحة الرئيسية

## التشغيل

1. افتح `index.html` في المتصفح لعرض الموقع
2. اضغط على أيقونة التروس (أسفل يمين الصفحة) للوصول للوحة التحكم
3. سجّل الدخول وابدأ التعديل

## الملفات

- `index.html` — الصفحة الرئيسية
- `admin.html` — لوحة التحكم
- `css/styles.css` — تنسيقات الموقع
- `css/admin.css` — تنسيقات لوحة التحكم
- `js/main.js` — سكريبت الموقع
- `js/admin.js` — سكريبت لوحة التحكم

## ملاحظة

التعديلات تُحفظ في **localStorage** و**sessionStorage** في المتصفح. للتشغيل مع قاعدة بيانات حقيقية، يُنصح بربط الموقع بخادم وخزن البيانات في قاعدة بيانات.
