# IT NORD — موقع الحلول التقنية

موقع عصري لشركة IT NORD مع قاعدة بيانات SQLite ولوحة تحكم بدون تسجيل دخول.

## المميزات

- **قاعدة بيانات SQLite** — خزن دائم لجميع المحتوى
- **خادم Python (Flask)** — API وخدمة الموقع
- **لوحة تحكم بدون تسجيل دخول** — إدارة مباشرة للمحتوى
- **مشاريع تعليمية** — تصميم التطبيقات، إنشاء السيرفرات، إنشاء المواقع
- **صفحة منفصلة لكل مشروع** — كل مشروع له نافذة/صفحة مستقلة (ليس على الصفحة الرئيسية)

## التشغيل

1. **تثبيت المتطلبات:**
   ```bash
   pip install -r requirements.txt
   ```

2. **تشغيل الخادم:**
   ```bash
   python app.py
   ```

3. **افتح في المتصفح:**
   - الموقع: http://127.0.0.1:5000
   - لوحة التحكم: http://127.0.0.1:5000/admin

## الملفات

| الملف | الوصف |
|-------|-------|
| `app.py` | خادم Flask وقاعدة البيانات |
| `index.html` | الصفحة الرئيسية |
| `admin.html` | لوحة التحكم (بدون تسجيل دخول) |
| `project.html` | صفحة عرض مشروع واحد |
| `data/itnord.db` | قاعدة بيانات SQLite (تُنشأ تلقائياً) |
| `css/` | التنسيقات |
| `js/main.js` | سكريبت الموقع |
| `js/admin.js` | سكريبت لوحة التحكم |

## لوحة التحكم

- **الصور والفيديو** — تغيير صور الأقسام
- **البطاقات المميزة** — تعديل بطاقات الصفحة الرئيسية
- **الأدوات** — إدارة أدواتنا وتقنياتنا
- **المشاريع** — إضافة/تعديل/حذف المشاريع التعليمية (تصميم التطبيقات، السيرفرات، المواقع)
- **أزرار التواصل** — إدارة أزرار الهاتف، واتساب، البريد

## API

- `GET/POST /api/features` — البطاقات المميزة
- `GET/POST /api/tools` — الأدوات
- `GET/POST /api/media` — الصور
- `GET/POST /api/comm` — أزرار التواصل
- `GET/POST /api/projects` — المشاريع
- `GET /api/projects/<id>` — مشروع واحد
